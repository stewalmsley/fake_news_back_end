Steve News - Demo News API Documentation

Users

GET https://steve-news.herokuapp.com/api/users - Provides information on all users GET https://steve-news.herokuapp.com/api/users/:username - Provides information on a user, their articles and comments

Topics

GET https://steve-news.herokuapp.com/api/topics - Provides information on all topics
GET /api/topics/:topic_slug/articles - Returns all the articles for a certain topic - the slug is a lowercase topic name
POST /api/topics/:topic_slug/articles Add a new article to a topic. This route requires a JSON body with title and body key value pairs # e.g: `{ "title": "new article", "body": "This is my new article content", "created_by": "user_id goes here"}`

Articles

GET /api/articles - Returns all articles
GET /api/articles/:article_id
Returns an individual article by the MONGO ID
GET /api/articles/:article_id/comments
Returns all the comments for a individual article, using the article MONGO ID
POST /api/articles/:article_id/comments
Add a new comment to an article. This route requires a JSON body with body and created_by key value pairs e.g: `{"body": "This is my new comment", "created_by": "user_id goes here"}`
PATCH /api/articles/:article_id
Increment or Decrement the votes of an article by one. This route requires a vote query of 'up' or 'down' e.g: `/api/articles/:article_id?vote=up`
DELETE /api/articles/:article_id
Delete an article by ID

Comments

PATCH /api/comments/:comment_id
Increment or Decrement the votes of a comment by one. This route requires a vote query of 'up' or 'down' e.g: `/api/comments/:comment_id?vote=down`
DELETE /api/comments/:comment_id Deletes a comment
